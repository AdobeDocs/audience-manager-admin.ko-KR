# Tableau 사용자에게 포털 사용자 할당 {#assign-a-portal-user-to-tableau-user}

<!-- t_tabeau.xml -->

[!UICONTROL Reporting] 페이지를 사용하여 포털 사용자를 [!DNL Tableau] 사용자로 만드십시오. 이렇게 하면 사용자가 Audience Manager에서 [!DNL Tableau] 보고서를 볼 수 있습니다.

1. **[!UICONTROL Reporting]** > **[!UICONTROL Assign Portal User to Tableau User]**&#x200B;을(를) 클릭합니다.

   ![](assets/tableau.png)

1. 사용자를 할당하려면 원하는 파트너 행에서 텍스트 필드에 [!DNL Tableau] 사용자 이름을 입력한 다음 텍스트 필드 외부를 클릭합니다.

사용자 지정을 삭제하려면 원하는 파트너 행에서 텍스트 필드에서 사용자 이름을 제거한 다음 텍스트 필드 외부를 클릭합니다.
